### What I learned

How to use grid to design or place your element at your destined location.

Here is a little example:

```css
.page{
        display: grid;
        grid-template-columns: 250px  250px 250px 250px;
        grid-template-rows: 280px 280px;
        column-gap: 20px;
        row-gap: -20px;
        align-items: center;
        justify-items: center;
        width: 1050px;
    }
```
